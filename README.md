# Excel Data Analysis Portfolio

Welcome to my repository of Excel projects! This collection is designed to showcase my proficiency in transforming raw, unstructured data into meaningful insights and automated solutions using Microsoft Excel.

##  Repository Overview

This portfolio contains a variety of projects that demonstrate end-to-end data workflows, from initial acquisition and cleaning to final visualization and automation. Each project highlights different skills and tools within the Excel ecosystem.

## Skills Demonstrated

Here are the core competencies you will find evidence of in this repository:

*  **Data Cleaning & Transformation:**
    *   Power Query (Get & Transform) for data import and shaping.
    *   Removing duplicates, handling missing data, and standardizing formats.
    *   Unpivoting data and merging multiple sources.

*   **Data Analysis & Modeling:**
    *   Advanced Excel formulas and functions (XLOOKUP, INDEX/MATCH, SUMIFS, etc.).
    *   PivotTables and PivotCharts for dynamic data summarization.
    *   Data Model creation and DAX formulas for more complex analysis.

*   **Data Visualization & Reporting:**
    *   Building interactive and user-friendly Dashboards.
    *   Using conditional formatting and sparklines for effective data storytelling.
    *   Creating a variety of charts to best represent underlying trends.

*   **Automation & Efficiency:**
    *   Excel VBA macros to automate repetitive tasks.
    *   Creating custom functions and automating data refresh processes.
    *   Designing user forms for simplified data entry.
